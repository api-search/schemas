description: Represents a financial transaction in Temenos Transact (formerly T24) core banking system. A transaction is a debit or credit entry posted against a customer account, capturing the movement of funds with full audit trail including booking date, value date, narrative, counterparty details, and applied charges or exchange rates. Transactions are generated by various banking operations including fund transfers, payment orders, standing orders, direct debits, interest postings, and fee collections.

properties_list:
- description: Unique system-generated transaction identifier assigned by Temenos Transact. Follows the format prefix plus numeric sequence (e.g., FT for fund transfers, PI for payment instructions).
  name: transactionId
  type: string
- description: Identifier of the account against which the transaction is posted. References an account arrangement created through the Arrangement Architecture.
  name: accountId
  type: string
- description: Classification of the transaction type indicating the nature of the financial operation that generated this entry.
  name: transactionType
  type: string
- description: Indicates whether the transaction is a debit (reducing account balance) or credit (increasing account balance) from the perspective of the account holder.
  name: debitOrCredit
  type: string
- description: Transaction amount in the account currency. Always expressed as a positive value; the debitOrCredit field indicates the direction.
  name: amount
  type: number
- description: ISO 4217 currency code of the transaction amount. Typically matches the account currency unless a foreign currency transaction is involved.
  name: currency
  type: string
- description: The date the transaction was booked (posted) to the account ledger. This is the date the transaction appears on the account statement.
  name: bookingDate
  type: string
- description: The value date used for interest calculation purposes. May differ from the booking date, particularly for future-dated payments or backdated corrections.
  name: valueDate
  type: string
- description: The date the transaction was processed by the system. Typically matches the booking date but may differ for batch-processed transactions.
  name: processingDate
  type: string
- description: Transaction reference number used for identification and reconciliation. May be system-generated or provided by the originator.
  name: reference
  type: string
- description: End-to-end reference that travels with the payment through the entire processing chain. Used for payment tracking and reconciliation across systems.
  name: endToEndReference
  type: string
- description: Human-readable description of the transaction providing context about the payment purpose or origin.
  name: narrative
  type: string
- description: Details of the other party involved in the transaction, such as the sender (for credits) or recipient (for debits).
  name: counterparty
  type: object
- description: Running balance of the account after this transaction was applied. Provides a sequential view of account balance movement.
  name: balance
  type: number
- description: Exchange rate information when the transaction involves currency conversion.
  name: exchangeRate
  type: object
- description: Charges and fees applied to this transaction.
  name: charges
  type: object
- description: Identifier of the account statement period this transaction belongs to.
  name: statementNumber
  type: string
- description: Indicates whether this transaction is a reversal of a previously posted transaction.
  name: reversalIndicator
  type: boolean
- description: Reference to the original transaction identifier when this entry is a reversal or correction.
  name: originalTransactionId
  type: string
- description: Lifecycle status of the transaction within the Temenos Transact processing pipeline.
  name: transactionStatus
  type: string
- description: The banking channel through which the transaction was initiated.
  name: channel
  type: string
- description: The payment method or instrument used for this transaction.
  name: paymentMethod
  type: string
- description: Transaction category code used for accounting and reporting classification within Temenos Transact.
  name: categoryCode
  type: string
- description: Audit trail information for the transaction.
  name: audit
  type: object
