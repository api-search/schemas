# API Search - Schemas

This repository stores JSON Schema references for the [APIs.io](https://apis.io) search network.

## About

Schema records document the data models used by APIs indexed by the network. They are generated from api-evangelist repos and used to build the `_schemas/` Jekyll collection on the APIs.io network site.

## Related Repositories

- [apis](https://github.com/api-search/apis) - Individual API records
- [providers](https://github.com/api-search/providers) - API provider records
- [capabilities](https://github.com/api-search/capabilities) - Workflow and capability definitions
- [network](https://github.com/api-search/network) - The APIs.io network site
